
# LUXA E-Commerce Platform

A ecommerce start-up designed to transform online shopping by enhancing transparency, trust, and customer experience through interactive 3D product visualization and verified vendor visibility,coupled with secure escrow payment system

## 🚀 Features
- Secure authentication
- Product management system
- Payment integration with ESCROW!
- Responsive frontend
- 3D Product viewing!!

## 🛠️ Tech Stack
- Python / Django
- HTML, CSS, JS
- PostgreSQL (SQLite for dev)
- Bootstrap
- Node.js, three.js
